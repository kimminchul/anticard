`<FAQ>`는 자주 묻는 질문 영역입니다. native `<details>` / `<summary>` 사용 — JS 없이 동작, 접근성·성능 모두 좋음.

```tsx
import { FAQ } from "@freeive/anti-card";

<FAQ items={[
  { question: "어디에 사용하나요?", answer: "랜딩 / 콘텐츠 사이트.", defaultOpen: true },
  { question: "shadcn과 다른가요?", answer: "정반대 방향입니다." },
]} />
```

## Props

| Prop | Type | Description |
|---|---|---|
| `items` | `FAQItem[]` | `{question, answer, defaultOpen?}[]` |

## 디자인

- dl + 각 행 `<details>`
- divide-y + border-y (헤어라인 행 구분)
- summary hover시 emerald
- + 아이콘 → group-open시 45deg 회전 (×)
- 답변은 14.5px leading-relaxed body 톤

## 접근성

- `<details>` native — 키보드·스크린리더 자동 지원
- defaultOpen=true 시 SSR에서도 열림 상태로 렌더
