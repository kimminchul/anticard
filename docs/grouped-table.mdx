`<GroupedTable>`은 [`<DataTable>`](/anti-card/data-table) 위에 **카테고리·섹터 그룹 헤더**를 자동 삽입하는 wrapper입니다. `groupBy` 함수로 행을 그룹화하고, 그룹 시작마다 헤더 행을 끼워 넣습니다.

```tsx
import { GroupedTable } from "@freeive/anti-card";

<GroupedTable<Post>
  data={posts}
  groupBy={(p) => p.category}
  collapsible
  columns={[
    { key: "title", header: "제목" },
    { key: "publishedAt", header: "발행일", align: "right" },
  ]}
/>
```

## Props

`<DataTable>`의 prop 중 `data` / `rowKey` / `columns` / `onRowClick` / `selection` / `expansion` / `rowClassName` / `getCellSpan`은 wrapper가 자체 관리.

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `T[]` | — | 행 데이터 |
| `groupBy` | `(row) => string` | — | **필수** — 그룹 키 추출 |
| `columns` | `DataTableColumn<T>[]` | — | 표 컬럼. 그룹 컨텍스트에서 sortable 자동 비활성 |
| `renderGroupHeader` | `(group, rows) => ReactNode` | `"group · N건"` | 헤더 커스터마이즈 |
| `rowKey` | `(row, i) => string` | `"{group}-{i}"` | 행 unique key |
| `collapsible` | `boolean` | `false` | 그룹별 펼침/접힘 활성 (chevron) |
| `defaultCollapsedGroups` | `string[]` | `[]` | 초기 접힌 그룹 (uncontrolled) |
| `collapsedGroups` | `string[]` | — | controlled 접힌 그룹 |
| `onCollapsedChange` | `(groups: string[]) => void` | — | 접힘 변경 콜백 |

## 디자인

- 그룹 헤더 행은 `DataTable`의 `rowClassName` + `getCellSpan` 활용:
  - **rowClassName**: `border-t border-zinc-300 bg-zinc-100/50` (그룹 시작 시각 분리)
  - **getCellSpan**: 첫 컬럼이 `columns.length`만큼 colSpan으로 전체 행 차지
- 헤더 라벨: `11.5px uppercase smallcaps zinc-500` + count `font-mono tabular-nums`
- collapsible 시 `lucide ChevronRight` (펼침 시 `rotate-90`)

## 등장 순서 보존

`groupBy`가 같은 키로 묶인 행들의 등장 순서를 그대로 유지 (`Map` 기반). 정렬·필터된 순서로 그룹화하고 싶다면 부모에서 `data`를 미리 정렬.

## 커스텀 헤더

```tsx
<GroupedTable
  groupBy={(p) => p.category}
  renderGroupHeader={(group, rows) => (
    <>
      {group}
      <span className="ml-3 text-emerald-600">
        ({rows.filter((r) => r.featured).length}개 featured)
      </span>
    </>
  )}
  ...
/>
```
