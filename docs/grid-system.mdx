`<GridSystem>` + `<GridCol>`은 **12 column grid system**입니다. `<Grid>`(균등 1~6열)와 다름 — 자식이 자유롭게 `col-span`을 지정.

```tsx
import { GridSystem, GridCol } from "@freeive/anti-card";

<GridSystem>
  <GridCol span={3}>좌측 sidebar</GridCol>
  <GridCol span={9}>본문</GridCol>

  <GridCol span={4}>1/3</GridCol>
  <GridCol span={4}>2/3</GridCol>
  <GridCol span={4}>3/3</GridCol>
</GridSystem>
```

## Props

### GridSystem

| Prop | Type | Default | Description |
|---|---|---|---|
| `columns` | `6 \| 8 \| 12 \| 16 \| 24` | `12` | 전체 컬럼 수 |
| `gap` | `"tight" \| "default" \| "loose"` | `"default"` | 간격 |
| `as` | `"div" \| "section"` | `"div"` | 시맨틱 |

### GridCol

| Prop | Type | Default | Description |
|---|---|---|---|
| `span` | `number (1~12)` | `12` | 데스크톱 col-span |
| `start` | `number (1~12)` | — | 시작 컬럼 (offset) |
| `as` | `"div" \| "section" \| "article" \| "aside"` | `"div"` | 시맨틱 |

## Grid vs GridSystem

| | Grid | GridSystem |
|---|---|---|
| 컬럼 결정 | 부모 (columns prop) | 부모 12 + 자식 span |
| 분할 | 균등만 | 자유 (3+9, 8+4 등) |
| 용도 | 이미지·작은 카드 균등 | 사이드바+본문, asymmetric |
| 컴포넌트 | `<Grid>` 단일 | `<GridSystem>` + `<GridCol>` |

**원칙**:
- 균등 정렬이면 `<Grid>` (단순)
- 자유 layout(3+9, 8+4 등)이면 `<GridSystem>` + `<GridCol>`

## 자주 쓰는 패턴

### 1. 사이드바 + 본문 (3+9)
Heritage / Lab의 sectors 패턴.
```tsx
<GridSystem>
  <GridCol span={3}>섹터 라벨</GridCol>
  <GridCol span={9}>프로젝트 리스트</GridCol>
</GridSystem>
```

### 2. 블로그 (article + aside, 8+4)
```tsx
<GridSystem gap="loose">
  <GridCol span={8} as="article">본문</GridCol>
  <GridCol span={4} as="aside">사이드</GridCol>
</GridSystem>
```

### 3. 가운데 정렬 (6/12, 가운데)
```tsx
<GridSystem>
  <GridCol span={6} start={4}>가운데 영역</GridCol>
</GridSystem>
```

## 모바일

`<GridSystem>`은 모바일(sm 이하)에서 자동 1열 stack — 모든 GridCol이 행 전체를 차지. 좌측·우측 분할이 위·아래 stack으로 자연스럽게 변환.
