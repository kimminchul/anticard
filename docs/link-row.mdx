`<LinkRow>`는 안티 카드의 **표준 CTA**입니다. shadcn Button처럼 채워진 박스 대신 한 줄 텍스트 + 화살표 + hover 시 emerald 색만 변하는 가벼운 형태.

```tsx
import { LinkRow } from "@freeive/anti-card";

<LinkRow href="/heritage">전체 Heritage 보기</LinkRow>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `href` | `string` | — | 링크 경로 (필수) |
| `external` | `boolean` | `false` | true면 target='_blank' + rel 자동 |
| `trailing` | `ReactNode` | — | 우측 보조 라벨 (smallcaps) |
| `tone` | `"default" \| "accent"` | `"default"` | hover시 emerald / 항상 emerald |
| `size` | `"default" \| "large"` | `"default"` | 15.5px / 18~20px |
| `className` | `string` | — | Tailwind 클래스 추가 |

## tone 가이드

- **default** (70% 사용): 본문 톤 + hover 시 emerald — 섹션 끝 "전체 보기 →" 등 일반 CTA
- **accent** (30% 사용): 항상 emerald — Talk·메인 CTA처럼 강조 필요한 곳

## external

```tsx
<LinkRow href="https://github.com/freeive" external trailing="외부">
  GitHub
</LinkRow>
```

`external=true`면 `target="_blank"` + `rel="noopener noreferrer"` 자동.

## HTML 레퍼런스

```html
<a href="/heritage" class="group inline-flex items-baseline gap-2 text-[15.5px]
   text-zinc-900 dark:text-zinc-100 hover:text-emerald-600 dark:hover:text-emerald-400
   transition-colors">
  <span class="border-b border-current/30 group-hover:border-current">전체 Heritage 보기</span>
  <span aria-hidden class="transition-transform group-hover:translate-x-0.5">→</span>
</a>
```

## 안티 패턴

```html
<!-- ❌ 채워진 박스 버튼 (shadcn 톤) -->
<a class="rounded-md bg-emerald-600 text-white px-4 py-2 font-semibold shadow">
  전체 Heritage 보기
</a>
```

→ LinkRow가 이 톤을 거부합니다. CTA는 박스가 아니라 **한 줄 텍스트의 색 변화**.
