`<Lead>`는 헤딩 직후의 인트로 본문입니다. 본문보다 한 단계 흐린 회색이라 별도의 박스나 borders 없이도 위계가 자연스럽게 만들어집니다.

```tsx
import { Lead } from "@freeive/anti-card";

<HeroHeading>제목</HeroHeading>
<Lead>한두 문단의 인트로 카피.</Lead>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `"default" \| "large"` | `"default"` | 15px 또는 18~20px |
| `width` | `"narrow" \| "default" \| "wide" \| "full"` | `"default"` | 44ch / 58ch / 72ch / 무제한 |
| `className` | `string` | — | Tailwind 클래스 추가 |

## 크기 가이드

| 사용처 | size | 이유 |
|---|---|---|
| `<SectionHeading>` 다음 | `default` (15px) | 일반 섹션 본문 톤 |
| `<HeroHeading>` 다음 | `large` (18~20px) | 큰 제목과의 균형 |
| 사이드 / 푸터 | `default` + `width="narrow"` | 좁은 너비 |

## 너비 가이드 (한 줄 글자 수)

- `narrow` 44ch — 좁은 영역
- `default` 58ch — 황금 너비 (Bringhurst 65~75자)
- `wide` 72ch — 본문 길이 강조
- `full` — 카드 같은 좁은 컨테이너 안에서

## HTML 레퍼런스

```html
<p class="text-[15px] leading-relaxed text-zinc-600 dark:text-zinc-300 max-w-[58ch]">
  한두 문단의 인트로 카피.
</p>

<!-- large -->
<p class="text-[18px] md:text-[20px] leading-relaxed text-zinc-600 dark:text-zinc-300 max-w-[58ch]">
  AI가 만들면 다 비슷해진다고 누가 정했나.
</p>
```
