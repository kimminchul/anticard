`<Quote>`는 본문 안의 인용구입니다. 박스로 가두지 않고 **좌측 헤어라인 1~2px만**으로 위계를 만듭니다.

```tsx
import { Quote } from "@freeive/anti-card";

<Quote cite="— Bringhurst, The Elements of Typographic Style">
  한 줄에 65~75자가 가장 읽기 좋다.
</Quote>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `cite` | `ReactNode` | — | 출처 (figcaption smallcaps) |
| `size` | `"default" \| "large"` | `"default"` | lead / leadLarge 토큰 |
| `className` | `string` | — | Tailwind 클래스 추가 |

## 디자인

- `<figure>` + `<blockquote>` + `<figcaption>` 시맨틱
- blockquote: `border-l-2 border-zinc-300 dark:border-white/[0.12]` + `pl-5`
- 본문 색: lead 토큰 (zinc-600 dark:zinc-300, 한 단계 흐림)
- cite: 12px uppercase tracking-[0.08em] zinc-500

## size

- `default` — 일반 본문 인용 (15px)
- `large` — 글의 핵심 인용 (18~20px). 글당 1~2번만

## HTML 레퍼런스

```html
<figure class="my-8">
  <blockquote class="border-l-2 border-zinc-300 dark:border-white/[0.12] pl-5
                     text-[15px] leading-relaxed text-zinc-600 dark:text-zinc-300">
    한 줄에 65~75자가 가장 읽기 좋다.
  </blockquote>
  <figcaption class="mt-3 pl-5 text-[12px] uppercase tracking-[0.08em]
                     text-zinc-500 dark:text-zinc-400">
    — Bringhurst
  </figcaption>
</figure>
```
