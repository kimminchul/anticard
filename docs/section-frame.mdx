`<SectionFrame>`은 안티 카드 5원칙 중 **공간(여백) + 헤어라인 + smallcaps 라벨**
세 가지를 한 컴포넌트로 묶어, AI가 카드 박스를 생성하려는 충동을 줄입니다.

```tsx
import { SectionFrame } from "@freeive/anti-card";

<SectionFrame
  eyebrow="Library · v0"
  title="카드 안에 카드를 쌓지 않는다."
  description="AI 도구가 생성하는 동질화 UI에 대한 답변."
>
  {/* 본문 콘텐츠 */}
</SectionFrame>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `eyebrow` | `ReactNode` | — | 섹션 카테고리 라벨 (smallcaps) |
| `title` | `ReactNode` | — | 섹션 큰 제목 |
| `description` | `ReactNode` | — | 제목 아래 서브 카피 |
| `as` | `"h1" \| "h2" \| "h3"` | `"h2"` | 헤딩 레벨 |
| `divider` | `boolean` | `true` | 위쪽 헤어라인 표시 여부 |
| `children` | `ReactNode` | — | 본문 (리스트, 그리드, ListRow 등) |
| `className` | `string` | — | Tailwind 클래스 추가 |
| `...rest` | `HTMLAttributes<HTMLElement>` | — | (`title` 제외) |

## 헤딩 레벨

페이지의 첫 큰 섹션이면 `as="h1"`, 아니면 기본 `h2`. 하위 영역이면 `h3`.

```tsx
<SectionFrame as="h1" eyebrow="Hero" title="..." />
<SectionFrame eyebrow="Pillars" title="..." />
<SectionFrame as="h3" eyebrow="Sub" title="..." />
```

## 헤어라인 끄기

연속된 섹션의 첫 번째에서는 헤어라인을 빼는 게 자연스럽습니다.

```tsx
<SectionFrame divider={false} eyebrow="Hero" title="..." />
<SectionFrame eyebrow="Pillars" title="..." />     {/* 위쪽 라인 자동 */}
```

## 디자인 메모

- 위·아래 패딩 `py-16 md:py-20` (안티 카드 5원칙 중 "공간")
- 헤어라인은 `border-t border-zinc-200/60 dark:border-white/[0.06]`
- `data-anti-card="section-frame"` attribute로 외부 스타일 타겟팅 가능

## 활용 사례

```tsx
<SectionFrame
  eyebrow="Sectors"
  title="어떤 무대에서 일했는가."
>
  <ul className="divide-y divide-white/[0.04] border-y border-white/[0.04]">
    <ListRow meta="2024" trailing="EBS">EBS 온라인 클래스 재구조화</ListRow>
    <ListRow meta="2023" trailing="롯데카드">mydata 수집 및 admin 개발</ListRow>
  </ul>
</SectionFrame>
```

## HTML 레퍼런스

React 안 쓰는 환경에서 같은 톤. 그대로 복사:

```html
<section class="border-t border-white/[0.06] py-16 md:py-20">
  <p class="text-[12px] uppercase tracking-[0.08em] text-zinc-500 font-medium">
    Heritage · 2016 — Now
  </p>
  <h2 class="mt-3 max-w-[20ch] text-2xl md:text-3xl font-semibold tracking-tight text-zinc-50">
    어떤 무대에서 일했는가.
  </h2>
  <p class="mt-6 max-w-[58ch] text-[15px] leading-relaxed text-zinc-300">
    텔레콤·금융·교육의 큰 싸움을 거친 깊이가 토대.
  </p>
  <div class="mt-10 md:mt-12">
    <!-- children: ul + li 또는 자유 콘텐츠 -->
  </div>
</section>
```

페이지 첫 섹션이면 `border-t` 부분 제거 (= `divider={false}` 효과).

Hero 섹션이면 `py-24 md:py-32` 로 더 큰 여백, `<h2>` 대신 `<h1>`,
`text-[clamp(2rem,5vw,4rem)] leading-[1.05]` 로 큰 헤딩.
